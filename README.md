
# webpack-react-ssr

## 开发模式

### 启动服务

```sh
npm run start
```

入口: http://127.0.0.1:3000/html/index.html 

### 查看编译输出

```sh
npm run build
```

## 生成模式

### 编译

```sh
npm run build-prod
```

### 启动服务

```sh
npm run start-prod
```

入口: http://127.0.0.1:3000/html/index.html 

## ESLint

### 检查

```sh
npm run lint
```

### Fix

```sh
npm run fix
```
